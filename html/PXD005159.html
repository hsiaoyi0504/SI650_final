Dataset Summary
TitleHeLa standard shotgun DDA analysis using a two-meter C18 monolithic silica column
DescriptionHuman HeLa proteome analysis by a nanoLC-MS/MS system using a two-meter C18 monolithic silica column under shallow gradient conditions without pre-fractionation was performed in this project. A hybrid quadrupole-orbitrap mass spectrometer (Thermo Q-Exactive) was used to analyze a tryptic digest of HeLa cells through this study. Identification results were filtered at 1% false discovery rate (FDR) thresholds by searching against a randomized decoy database using Mascot at peptide and protein levels.
