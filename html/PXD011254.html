Dataset Summary
TitleLOPIT-DC subcellular map of U-2 OS cells
DescriptionGeneration of a high-resolution subcellular localisation map for the proteome of the human cancer cell line U-2 OS, using the newly developed LOPIT-DC method. Comparison of the results generated using LOPIT-DC with data generated using the well-established hyperLOPIT method and data mining regarding multi-localising proteins, signalling pathways, isoforms and large protein complexes. Integration of the two methods using transfer learning.
