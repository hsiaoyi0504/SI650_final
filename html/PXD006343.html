Dataset Summary
TitleMetaproteomics of phototrophic biomats from two soda lakes in the Canadian Rocky Mountains
DescriptionThis data is a case study done in the context of developing methods for assessing the taxonomic composition of microbial communities using metaproteomics. For this study with analyzed phototrophic biomats from two Soda Lakes in the Canadian Rocky Mountains using metaproteomics. For protein identification we generated a metagenome from which we predicted and annotated the protein sequences used to analyze the metaproteomes. The database is available in this PRIDE submission.
